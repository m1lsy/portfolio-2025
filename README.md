# Portfolio 2025

Starter portfolio for Mila Pulvirenti — a minimal HTML/CSS/JS scaffold.

To view locally, open `index.html` in your browser or run a simple static server, e.g.:

```bash
# macOS / Node (python alternative below)
npx http-server . -c-1

# or with Python 3
python3 -m http.server 8000
```

Files added:
- `index.html` — homepage
- `case-study.html` — sample case study
- `css/styles.css` — styles
- `js/script.js` — tiny JS
- `assets/*.svg` — simple placeholder graphics
